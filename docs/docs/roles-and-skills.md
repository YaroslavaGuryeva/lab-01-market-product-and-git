## Roles and responsibilities
1. Embedded Developer
   - Firmware Development: Write low-level code
   - Device Driver Development: Create and maintain software that allows the OS to communicate with hardware
   - Real-Time Systems Programming: Develop software with strict timing constraints
   - Performance Optimization: Ensure code is highly efficient in terms of memory usage and processing speed
2. Backend Developer
   - Creation of product's key functionality 
   - Managing connection to DB
4. Information Security Engineer
   - managing vulnerabilities
   - reporting detected pitfalls
   - creating new ways of secure program work
6. DevSecOps
   - controlling the workflow of product development
   - managing security issues
8. QA
   - reporting detected pitfalls
   - testing the product functionality 

## Common skills across roles
The common skills are creating product functionality, managing its security,
programming language knowledge, knowledge of version control systems.
